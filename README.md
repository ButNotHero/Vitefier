# Vitefier

Vite SSR template.

## Included
- i18n
- Windi CSS
- Vue 3
- File based routing
- Components auto importing
- Layout system
- SSR
- TypeScript
- Dark/Light themes
- ESLint
- Stylelint
- Commitlint
- Axios
- Pinia

## Usage

### Development

Just run and visit http://localhost:3333

```bash
yarn run dev
```
Without SSR:

```bash
yarn run dev:spa
```

### Linters

Fix all files

```bash
yarn run fix
```

### Build

To build the App, run

```bash
yarn run build
```

### Start

Run production Node server

```bash
yarn run server
```

### Auto run production

To build the App, run

```bash
yarn run start
```
